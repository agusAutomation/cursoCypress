# cursoCypress

Para correr los tests emplear el comando

npx cypress open

Otro comando posible es:

cypress -v para ver la version

## To run

`npm run cy:open:test:desktop`

## To Visual Tetsing

Run the script
`cy:open:test:desktop:generate:baseImage`
